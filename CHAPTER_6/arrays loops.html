What is an Arrays

  1. An Array is just a list of values.
  2. index:Starts with 0.
  3. Array are used for stroing multiple values in a single variable.

Array Syntax & Values

  1. Use to create a new array.brackets enclose list of values.
  2. Arrays can be saved to a variables.
  3. Accessible Values:Use with index.
  4. Syntax:Rules:Breackets start and end the array,Values separate by commans.Can span multiple lines.
  5. Arrays can hold any value,including arrays.
  6. typeOf operator on Arrayn Returns object.

Array Properties & Methods

  1. Array.isArray() Checks if a variable is an array.
  2. Length property holds the size of the array.
  3. Arrays also use references like object.
  4. De-structuing also works for Arrays.

Common Methods

  • push/pop:Add or remove to end.
  • shift/unshift:Add or remove from front.
  • splice:Add or remove elements.
  • toString:Convert to string.
  • sort:Sort elements.
  • valueOf:Get array itself.

What is a Loop

  • Code that runs multiple times based on a condition.
  • Loops also alter the flow of execution,similar to functions.Functions:Reusable blocks of code,Loops:Repeated execution of code
  • Loops automate repettive tasks.
  • Types of Loops:for,while,do-while.
  • Iterations:Number of times the loop runs.

While Loop

  • Iterations:Number of times the loop runs.
  • Used for non-standard conditions.
  • Repeating a block of code while a condition is true.
  • Remember:Always include an update to avoid infinite loops.

Do While Loop

  • Executes block first,then checks condition.
  • Guaranteed to run at least one iteration.
  • Unlike While,first iteration is unconditional.
  • Don't forget to update condition to aviod infinite loops.

For Loop

  • Standard loop for running code multiple items.
  • Generally preferred for counting iterations.

Accumulator Pattern

  • A Pattern to accumulate values through looping.

Common Scenarious

  1. Sum all the numbers in any array.
  2. Create a modified copy of an array.

Break & Continue

  1. Break lets you stop a loop early,or break out of a loop.
  2. continue is used to skip one iteration or the current iteration.
  3. In while loop remember to do the increment manually before using continue.
